APPENDIX B — Voice Line Labels
------------------------------
VIP / body found (foundDeadBodyRadioLabel):
  V_CPR0042_KEEP    "A VIP's dead. Looks like an enemy attack."
  V_CPRGZ0040       "One of the officers inspecting the base is dead."
  V_CPRGZ0040_UNKEEP  Same, no "stay on alert" tail.
  V_CPR0231         "Found the body of a prisoner."

Hostage escape (hostageLostLabel):
  V_CPR0250         "The girl's disappeared."
  V_CPR0250_KEEP    Same + keep-searching tail.
  V_CPR0270         "The kid's missing." (EN only)
  V_CPR0270_KEEP    Same + keep-searching tail.
  V_CPR0271         "The kid's missing." (alt)
  V_CPR0271_KEEP    Same + keep-searching tail.
  V_CPR0230_KEEP    Male variant + keep-searching tail.
