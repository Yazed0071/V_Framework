      Mark a mission code as emergency (iDroid emergency list).
    SetEmergencyMissionPopup(title, body)                  -> bool  -- literal text
    SetEmergencyMissionPopupLangId(titleLabel, bodyLabel)  -> bool  -- LangId labels
    ClearEmergencyMissionPopupOverride()
      Overrides the single generic "you have emergency missions" popup (one slot; each call
      replaces the last). nil field = engine default; unknown LangId = default.

  HUD mission icon (upper-left "urgent_time" widget)
    ShowMissionIcon(title, body, time)
      title/body: LangId string, or nil for engine default. time: seconds (default 6).
      Unregistered labels show nothing; accent color is fixed by the engine preset.

  HUD UI palette (mb_col_palette color overrides)
    SetUiPaletteColor(key, r, g, b [, a])  -> bool   key: "COL-3" or its hash; r,g,b,a = 0..1
    RestoreUiPalette([key])                          one key (string/hash), or all
      Returns false if the key isn't found. The palette only exists in memory after the
      iDroid / owning UI has opened once -- don't call from Init(); call on first UI open.
      Each call patches loaded palette data AND registers an override for future lookups, so
      transient widgets pick it up on rebuild; already-built widgets keep their cached color
      until they rebuild (reopen the pause menu). For modded .fox2 palettes that share key
      names, set a unique anchor key first so calls target your palette.

Custom GameObject.SendCommand IDs
---------------------------------
GameObject.SendCommand(targetSpec, { id = "XXX", ...params }). Unknown IDs fall through to the
engine. Global commands ignore the target -- use a placeholder like {type="TppSahelan2",group=0,index=0}.

  id="SetSahelanPhase",  phase=<number>   -> nil     single-shot Exit/Enter (phase enum above)
  id="GetSahelanPhase"                    -> number
  id="SetEscapeState",   enable=<boolean> -> nil     mark a hostage taken-by-player (hostage gameObjectId)
  id="SetOccasionalChatList",     labels={...}  -> nil   replace pool (every soldier; {} restores vanilla)
  id="InsertToOccasionalChatList",labels={...}  -> nil   add to pool (accumulates)
  id="RemoveFromOccasionalChatList",labels={...}-> nil   suppress labels (vanilla and custom)
  id="ResetOccasionalChatList"                  -> nil   restore vanilla
    (occasional-chat IDs are wrapped by V_TppEnemy.* above; labels = hashes or strings)

  Example:
    local t = { type = "TppSahelan2", group = 0, index = 0 }
    GameObject.SendCommand(t, { id = "SetSahelanPhase", phase = TppSahelan2.SAHELAN2_PHASE_ALERT })
    GameObject.SendCommand(GameObject.GetGameObjectId("hostage_01"), { id = "SetEscapeState", enable = true })
